Level¶

class sunpy.net.vso.attrs.Level(value)[source] [edit on github]¶

Bases: sunpy.net.vso.attrs._VSOSimpleAttr

Specifies the data processing level to search for. The data processing level is specified by the instrument PI. May not work with all archives.

Parameters:
  • value (float or string) –
  • value can be entered in of three ways (The) –
  • May be entered as a string or any numeric type for equality matching ((1)) –
  • May be a string of the format '(min) - (max)' for range matching ((2)) –
  • May be a string of the form '(operator) (number)' where operator is ((3)) –
  • of (one) –
